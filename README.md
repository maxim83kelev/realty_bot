# RealtyKelev Bot

Telegram-бот для мониторинга новых объявлений недвижимости в Чехии.

## Что делает
- Парсит чешские сайты недвижимости в режиме реального времени
- Мгновенно отправляет уведомления пользователям по их фильтрам
- Фильтры: город, цена, тип недвижимости

## Стек
- Python 3.14
- aiogram 3.x
- PostgreSQL + asyncpg
- httpx + BeautifulSoup
- Playwright
- APScheduler

## Команды бота
- /start — запустить бота
- /filter — настроить фильтр
- /myfilter — посмотреть фильтр
- /stop — остановить уведомления
